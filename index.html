Crespoter Graphics Engine

A graphics engine made using C++ and OpenGL. It supports the following features:.

  • check Blinn-Phong Reflection Model
  • check Directional, Point and Spot Light sources
  • check Loading 3D models
  • check Shadows for all light sources
  • check Cubemaps
  • check Anti Aliasing
  • check Gamma Correction
